The server needs your Uber/Lyft session cookies to fetch prices.

How to get them:
1. Log into m.uber.com in Chrome
2. Open DevTools → Application → Cookies → m.uber.com
3. Copy the full cookie string from the network tab (F12 → Network → any request → Request Headers → Cookie)
4. Add it as UBER_COOKIES in your Railway environment variables
5. Do the same for ride.lyft.com → LYFT_COOKIES
